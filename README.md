# 🛒 E-Commerce Sales & Profit Performance Report — Power BI Project

A complete sales and profit analysis project for **E-Commerce Store (2024–2025)** built using **Microsoft Power BI** and **Excel**. The project involves cleaning raw data, building data models, writing DAX measures, and designing an interactive dashboard to uncover key business insights.

## 📊 Dataset Overview

The raw dataset contains **1,000 records** with **17 columns** including:

| Column | Description |
|--------|-------------|
| Order ID | Unique order identifier |
| Order Date | Date the order was placed |
| Ship Date | Date the order was shipped |
| Ship Mode | Standard / Express / Same Day / Economy |
| Customer Name | Customer identifier |
| Region | North / South / East / West |
| City | City of delivery |
| Category | Product category (Electronics, Clothing, etc.) |
| Sub-Category | Product sub-category |
| Product Name | Product model name |
| Quantity | Number of units ordered |
| Unit Price | Price per unit (INR) |
| Sales | Total sales value after discount |
| Discount | Discount percentage applied |
| Profit | Profit earned from the transaction |
| Payment Mode | UPI / Credit Card / Debit Card / COD |
| Status | Delivered / Shipped / Returned / Cancelled |

## 📈 Dashboard Visuals

The dashboard includes **8 interactive visuals** on a single-page layout:

| Visual | Description |
|--------|-------------|
| **KPI Card — Total Sales** | Overall revenue generated across all orders |
| **KPI Card — Total Profit** | Net profit earned across all transactions |
| **KPI Card — Total Quantity Sold** | Total units sold across all categories |
| **Line Chart** | Monthly & yearly sales trend (2024 vs 2025) |
| **Area Chart** | Quarterly and monthly profit trend |
| **Column Chart** | Sales distribution across North / South / East / West |
| **Bar Chart** | Top cities ranked by total sales value |
| **Clustered Column Chart** | Profit comparison across product sub-categories |

---

## 🔍 Key Insights

- 🏆 **West and South regions** contribute the highest share of total sales revenue
- 💻 **Electronics** (Laptops, Mobile Phones) drive the highest profit margins
- 📅 **Q4 months (Oct–Dec)** show a consistent sales spike — ideal for campaign planning
- 💳 **UPI and Credit Card** are the most preferred payment modes
- 🔻 Discount rates above **15%** significantly reduce overall profit margins
- 🚚 **Express shipping** is most frequently used in metropolitan cities

## 🚀 How to Use

1. Download both files (`Ecommerce_Sales_Dataset.xlsx` and the `.pbix` file)
2. Open `E-COMMERCE_SALES___PROFIT_PERFORMANCE_DASHBOARD.pbix` in Power BI Desktop
3. If prompted, update the data source path to point to `Ecommerce_Sales_Dataset.xlsx`
4. Click **Refresh** to load the data
5. Use the **City** and **Quantity** slicers to filter all visuals interactively
